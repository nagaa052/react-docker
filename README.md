 # iXamDrive front v2
 ## Description
 ReactをベースとしたiXamDriveのフロント:v2
 [create-react-app](https://github.com/facebook/create-react-app)をベースに作成

## Installation

### ローカル実行
```
$ docker-compose up
$ open http://localhost/
```

## How to use
### package 追加
docker-compose run node yarn add react-router-dom
